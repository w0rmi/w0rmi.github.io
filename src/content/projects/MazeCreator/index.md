## Project Overview

Hey there! This is basic maze generator and solver created in python3 using tkinter.

## Maze-Generator Algorithms

- **Depth-first search.**
- **Minimum spanning tree(Prim's Maze Generator).**

## Maze-Solver Algorithm

- **Depth-first search.**

## Requirements

- **Tkinter v8.6.**
- **Python3 v3.8.5**

## Building from source

```
$ python3 main.py
```
